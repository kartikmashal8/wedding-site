## 🖼️ How to Add Photos

### Father's Photo (`father.jpg`)
1. Take the photo you want to use
2. Rename it exactly to `father.jpg`
3. Place it inside the `assets/images/` folder
4. Replace the placeholder file that is already there

> ✅ Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`  
> If you use `.png` instead of `.jpg`, open `index.html` in a text editor  
> and change `father.jpg` → `father.png` (just that one word)

## 🚀 How to Host on GitHub Pages (Free Hosting)

Follow these steps exactly — no coding knowledge needed.

### Step 1 — Create a GitHub Account
- Go to [https://github.com](https://github.com)
- Sign up for a free account if you don't have one

### Step 2 — Create a New Repository
1. Click the **+** button at the top right → **New repository**
2. Name it: `wedding` (or anything you like, e.g. `shubham-shweta-wedding`)
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload Your Files
1. In your new repository, click **Add file** → **Upload files**
2. Drag and drop the entire contents of this `wedding-site/` folder:
   - `index.html`
   - `assets/` folder (with images inside)
3. Scroll down, write a commit message like `Add wedding website`
4. Click **Commit changes**

> ⚠️ Make sure the folder structure is preserved:
> ```
> index.html         ← at the root level
> assets/
>   images/
>     ganesha.png
>     father.jpg
> ```

### Step 4 — Enable GitHub Pages
1. Go to your repository → **Settings** tab
2. In the left sidebar, click **Pages**
3. Under **Branch**, select `main` (or `master`) → folder `/root`
4. Click **Save**
5. Wait 1–2 minutes

### Step 5 — Get Your Website Link
- GitHub will show a green box with your link:  
  `https://YOUR-USERNAME.github.io/wedding/`
- Share this link with your guests! 🎉
